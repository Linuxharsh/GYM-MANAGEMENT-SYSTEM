# GYM-MANAGEMENT-SYSTEM

A gym management system built with HTML, CSS, and JavaScript with Firebase integration.

## Features
- Admin dashboard for managing members and creating bills
- Member dashboard for viewing bills
- Firebase authentication and Firestore database

## Setup
1. Clone the repository
2. Configure Firebase credentials in `JS/firebase.js`
3. Run with a local server (e.g., `npx serve .`)
4. Open `http://localhost:3000`
